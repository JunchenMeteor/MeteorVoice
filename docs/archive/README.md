# Docs Archive

本目录保存历史交接、早期提示词和过期阶段资料。归档文档保留用于追溯决策，不作为当前开发入口。

当前开发入口：

- `docs/index.md`
- `docs/development-rules.md`
- `docs/project-structure.md`
- `docs/native-mobile-architecture-plan.md`

如果归档文档与当前入口文档冲突，MUST 以当前入口文档为准。

## Archive Layout

- `docs/archive/handoffs/`: 历史 phase handoff 文档。
- `docs/archive/plans/`: 已完成或过期的阶段计划，保留用于回溯，不作为当前执行入口。
- `docs/archive/prompts/`: 早期一次性开发提示词。
